# Лабораторна робота №3

## Використання кастомних жестів у React Native та стилізація інтерфейсу мобільного застосунку.

---
## 🗒️ Інструкція запуску проєкту
### Клонування репозиторію
```bash
git clone https://github.com/KokhanTetiana/MobileLabsRN2026.git
```
```bash
cd MobileLabsRN2026
```
Перехід у проєкт лабораторної
```bash
cd lab3
```
Встановлення залежностей
```bash
npm install
```
▶️ Запуск проєкту
```bash
npx expo start
```
---
## Функціонал застосунку

**Tap (звичайний клік)** → +1 очко  
- **Double Tap (подвійний клік)** → +2 очки  
- **Triple Tap (потрійний клік)** → +10 очок + виконання завдання  
- **Long Press (утримання 3 сек)** → +5 очок + виконання завдання  
- **Pan (перетягування об’єкта)** → переміщення об’єкта + виконання завдання  
- **Swipe вправо (Fling вправо)** → випадкові очки (0–9) + завдання  
- **Swipe вліво (Fling вліво)** → випадкові очки (0–9) + завдання  
- **Pinch (масштабування)** → виконання завдання
